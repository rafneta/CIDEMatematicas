Objetivo: Realizar ejercicios sobre Tema 2.

Tipo de trabajo: Equipos (dos o tres personas).

Problema 1

1.1 ¿Qué son y para que sirven los métodos numéricos?

1.2 Si se tiene una función \(f:\mathbb{R}\rightarrow\mathbb{R}\), y se quiere resolver el problema,

\[f(x) = 0 \tag{1}\]

surgen varios aspectos a comentar:

  1. Puede pasar que el problema no tenga solución.
  2. Puede suceder que si exista solución, pero que no se cuente con un método para encontrar la solución
  3. Puede pasar que el problema tenga solución y exista un método para encontrar la solución.

Bajo el segundo escenario, el método de bisección es una forma de resolver el problema de la Equation 1. Este método es aplicable bajo ciertas condiciones de la función1, pero no es el único método.

Investiga sobre el método numérico de punto fijo. En esta investigación, es necesario presentar bajo que condiciones este método puede ser aplicable para resolver la Equation 1, puedes incluir gráficas, resultados (teoremas, lemas), ejemplos, etc.

Problema 2

2.1 Realiza un código en R que implemente el algoritmo de bisección (puedes apoyarte de los códigos de los laboratorios correspondientes).

Este código debe considerar como elementos de entrada

  • La función \(f\) para la aproximación de una raíz
  • El intervalo de búsqueda inicial \([a,b]\)
  • El número \(n\) de elementos a generar para la sucesión \(\{p_n\}\)
  • Una tolerancia \(\epsilon>0\), este número servirá como una medida del intervalo de búsqueda, si el intervalo de búsqueda tendrá un longitud menor a la indicada en la tolerancia, el programa deja de genrar los elementos de la sucesión.

Este código debe considerar como elementos de salida.

  • El valor aproximado de la raiz, que será último elemento generado.
  • La grafica de la sucesión \(\{p_n\}\)
  • La gráfica de la función \(f\) en el intervalo inicial de busqueda.

2.2 Con el código anterior resuelve el problema

  • \(f(x) = cos(x)-x\)
  • \([-1,3]\)
  • \(n=100\)
  • \(\epsilon=0.0001\)

Problema 4

Los problemas relativos al dinero necesario para pagar una hipoteca de una casa durante un periodo fijo de tiempo requieren la fórmula

\[A=\frac{P}{i}\left[1-(1+i)^{-n}\right]\]

denominada ecuación de la anualidad ordinaria. En esta ecuación, \(A\) es el importe de la hipoteca, \(P\) es el importe de cada pago e \(i\) es la tasa de interés por periodo para \(n\) periodos. Supongamos que se necesita una hipoteca de 135000 dólares por una casa a 30 años y que los pagos máximos que puede realizar el cliente son de 1000 dólares mensuales. ¿Cuál será el interés más alto que podrá pagar?

4.1 Argumenta porqué es aplicable o no el algoritmo de bisección. En caso de ser aplicable resuelve el problema con ayuda de tu código.

4.2 Argumenta porqué es aplicable o no el algoritmo de punto fijo. En caso de ser aplicable resuelve el problema con ayuda de tu código.

Problema 5

Se tiene el problema

\[x^2+x-6 = 0\]

Es claro que este problema tiene solución y existe un procedimiento para encontrar la solución. En ocasiones a pesar de tener una solución analítica de un problema, se elige utilizar un método numérico para “aproximar” la solución de un problema, pues se mejora en eficiencia computacional o definitivamente es una necesidad.

5.1 Argumenta porqué es aplicable o no el algoritmo de bisección. En caso de ser aplicable resuelve el problema (encontrar las dos raíces) con ayuda de tu código.

5.2 Argumenta porqué es aplicable o no el algoritmo de punto fijo. En caso de ser aplicable resuelve el problema (encontrar las dos raíces) con ayuda de tu código.

5.3 Investiga una biblioteca de R que resuelva la Equation 1. Utilizala para resolver el problema (encontrar las dos raíces).

Instrucciones de entrega

Forma

  1. Todos los integrantes cargan el mismo trabajo

  2. El formato de entrega es: problema-solución, en orden ascendente.

  3. Se anota el enunciado completo del problema

  4. El formato de entrega es libre (imágenes, tablet, etc.).

  5. La actividad debe ser desplegada en one-note. Si la actividad no es visible dentro de one-note, no se revisa.

Fondo

Lo ideal es entregar con notación matemática la solución de cada problema. Pero si aún se complica la notación, puedes redactar, pero siempre considera entregar la mayor notación posible.
